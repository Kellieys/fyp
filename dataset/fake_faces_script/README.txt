--App.js--

This script is to collect simulated face images as dataset from
[https://thispersondoesnotexist.com/]

Procedure to run this scripts
1. Go to project directory of this folder in command prompt
2. Make sure this folder contain package.json
3. Type "npm install" in command prompt and hit Enter
4. Type "node app.js" in command prompt and hit Enter
5. The image downloaded will be stored in a folder named "images" in the directory

The FULL dataset collected by this script is stored in Google Drive
[https://drive.google.com/drive/folders/1HcdbJURG95Yb3bkzJeAbQUiHYAf40vw4?usp=sharing]
